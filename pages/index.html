Why learners choose GasBank

Purpose-built for medical students and interns, GasBank focuses on the concepts that matter most in the operating room.

Residency-ready content

Focused explanations and learning objectives help you retain what matters as you ramp up to residency.

Interactive practice modes

Toggle between Tutor and Exam modes, track performance, and drill into categories with powerful filters.

Offline-first by design

Run sessions locally with zero logins. Automatic and manual backups keep progress secure on your machine.

Accessible authoring tools

Built-in analytics

Dashboard insights highlight trends, incorrect answers, and flagged questions so you always know what to review next.

Quick start

GasBank desktop runs on your computer with a streamlined setup. Clone or download the project, install dependencies, and go.

  1. Install dependencies with npm install.
  2. Launch the desktop app using npm run start.
  3. Build production binaries any time with npm run make.
